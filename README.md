# Omni-channel Support Dashboard

A professional, high-performance interface for customer support operators, built with **Vue 3** and **TypeScript**. This project mimics a production-grade low-code platform for managing multi-channel communications (Telegram, WhatsApp, Email) in a single workspace.

---

## 🚀 Overview

This dashboard is designed to demonstrate modern frontend practices in building scalable, real-time communication tools. It features a reactive ticket management system, automated workflow rules (low-code logic), and a modular UI component library.

## 🛠️ Tech Stack

- **Framework:** [Vue 3](https://vuejs.org/) (Composition API with `<script setup>`)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **State Management:** [Pinia](https://pinia.vuejs.org/) (Modular stores)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Icons:** [Lucide Vue Next](https://lucide.dev/)
- **I18n:** [Vue-i18n](https://vue-i18n.intlify.dev/) (Multi-language support: EN/UA)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strict mode)

## ✨ Key Features

- **Real-time Ticket Feed:** Reactive list of incoming messages with priority levels and channel identification.
- **Low-Code Automation Builder:** A dedicated module to create "IF-THEN" rules for automated ticket handling.
- **Agent Workspace:** Integrated chat interface with internal notes and status management (Online/Away/Offline).
- **Multi-language Support:** Full UI localization using `vue-i18n`.
- **Responsive & Accessible:** Clean, enterprise-ready UI that adheres to accessibility standards.
